# Sleek To-Do App

A clean, slightly-opinionated To-Do List web application featuring a rich dark mode UI, smooth glassmorphic effects, and persistent local storage.

## Features ✨
- **Beautiful UI**: Modern dark aesthetic with sleek radial gradients and soft glassmorphism.
- **Smooth Animations**: Fluid transitions for hover states, and when adding, checking, or deleting tasks.
- **Local Persistence**: Tasks are automatically saved to your browser's `localStorage` so you never lose track.
- **Responsive & Lightweight**: Built strictly with Vanilla HTML, CSS, and JS. Zero bloat, zero dependencies.

## Usage 🚀
Simply open `index.html` in your favorite web browser to start organizing your day!

## Technologies
- HTML5
- CSS3 (Custom Variables, Flexbox, Keyframe Animations)
- JavaScript (DOM Manipulation, LocalStorage)
